## execute

first instal [gnuCobol](https://gnucobol.sourceforge.io/) v3+ to run.

To compile a cobol program.

```sh
cd /folderWhereprogramIsStore
cobc -free -x -o nameofoutputfile nameofprogramtocompile.cbl
./nameofoutputfile
```
